How it works

Identiti is an AI-powered personal knowledge graph. You chat with our AI, and it extracts your skills, values, goals, and personality traits into a Neo4j graph database with field-level encryption. You can search, edit, add, or delete any node. When you're ready, download your Memory Card — a portable markdown snapshot of your entire identity — and paste it into ChatGPT, Claude, or any AI to give it full context about who you are.

Features

Memory Graph

Skills, goals, values, traits — every piece of you becomes a node in a Neo4j knowledge graph. Relationships between them become edges. Your identity, mapped and visualized on an interactive canvas.

Chat to Build

Just talk. The AI asks the right questions and builds your graph from the conversation. No forms, no friction. Your profile builds itself as you chat.

Fully Editable

Add, rename, or delete any node or relationship. Search your entire graph instantly. It's your data — you have full control over every piece of it.

Portable Memory Card

Download a markdown snapshot of your entire profile. Paste it into ChatGPT, Claude, or any AI to give it instant, deep context about who you are, what you value, and how you think.

Bring Your Own Key

Use your own API key from Groq, OpenAI, or Anthropic. Your key goes directly to the provider — Identiti never stores it. Full transparency, zero vendor lock-in.

End-to-End Encrypted

All graph data is encrypted at rest with per-user Fernet keys derived via HKDF. Only you can read your data. Not us, not anyone else.

Tech Stack

React + Vite frontend with interactive canvas graph renderer. Flask (Python) backend API. Supabase for Google OAuth authentication. Groq LLaMA 3.3 70B via LiteLLM for AI. Neo4j graph database for the knowledge graph. Redis for conversation history and staging pipeline. Field-level encryption with cryptography library (Fernet + HKDF). Deployed on Render (backend) and Vercel (frontend).

Privacy and Security

Your data is encrypted at rest with per-user keys. BYOK users' API keys go directly to the LLM provider from the browser — they never touch Identiti's servers. Your personal data is never used to train models. You can edit or delete any node at any time. Nothing is permanent unless you commit it.
